Elementos em nível de bloco

Elemento em nível de bloco sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço (uma margem) antes e depois do elemento.

Um elemento de nível de bloco sempre ocupa toda a largura disponível (estende-se para a esquerda e para a direita tanto quanto possível).

Dois elementos de bloco comumente usados ​​são: < p > e < div >.

O elemento < p > define um parágrafo em um documento HTML.

O elemento < div > define uma divisão ou seção em um documento HTML.

Tipos de Entrada de Dados

















Form - Formulário




























name: Nome do campo, utilizado no envio dos dados.

value: Valor padrão do campo.

placeholder: Texto exibido dentro do campo até que o usuário comece a digitar.

required: Torna o campo obrigatório.

maxlength: Limita o número de caracteres inseridos.

disabled: Desabilita o campo, impedindo interação.

readonly: Torna o campo apenas leitura.
